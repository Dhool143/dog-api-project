# Dog API Project

This project is a simple web application that uses a public Dog API to display dog breeds and dog facts.

## Features
- Fetches a list of dog breeds
- Click a breed to view details
- Displays dog facts
- Uses fetch API with async/await

## Technologies Used
- HTML
- JavaScript
- Public Dog API (https://dogapi.dog)

## How to Run
Open index.html using Live Server or a local server.
